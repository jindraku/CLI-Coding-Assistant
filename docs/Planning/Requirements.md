# Requirements

## Functional
- accept natural language coding tasks
- operate as an autonomous agent
- discover and call MCP tools dynamically
- support the official filesystem server
- support one external MCP server
- support a custom local RAG MCP server
- support at least one local model and one cloud model
- visibly show tool usage in the CLI

## Non-functional
- planning-first repository structure
- clean modular architecture
- persistent vector DB
- readable code
- simple local setup for grading
